Paint Using JavaSwing

2025 Read more ↗

Paint Application built using Java Swing. Implements a basic drawing interface with various tools and features. You can opend different canvases, save your work, and export it as an image file. You can also edit images using filters and transformations, and the application supports layers for more complex compositions. On the latest iterations I added webcam use and real-time video processing features, allowing users to apply filters and effects to their webcam feed and capture snapshots or record videos with the applied effects. It can also record audio from the microphone and reproduce audio files, making it a versatile multimedia application. Designed to learn object-oriented programming concepts in Java and integrations with libraries and frameworks.
